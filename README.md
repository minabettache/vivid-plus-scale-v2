# VIVID+ Scale Edition v2

A premium, mobile-first Next.js loyalty and membership MVP for Vivid Smoke Shop & Lounge.

## Included
- Cinematic welcome screen
- Customer onboarding
- Persistent membership on the device using localStorage
- Stable member ID
- Premium digital Gold card
- Home, Offers, Rewards, Events, and Profile navigation
- Reusable component architecture
- Mobile-first responsive UI

## Run on Windows
1. Extract the ZIP.
2. Open the extracted project folder.
3. Click the File Explorer address bar, type `cmd`, and press Enter.
4. Run `npm install --registry=https://registry.npmjs.org/`
5. Run `npx next dev`
6. Open `http://localhost:3000`

## Important
This version is a polished frontend MVP. Customer data is stored only in the browser on the current device. A production launch still needs a secure backend, authentication, database, employee permissions, real QR tokens, legal consent records, and payment/wallet integrations.
